# Available iOS Fonts

All fonts built into iOS 17. Use the **PostScript name** in `template.json`. If a name is not recognized, iOS falls back to the system font (San Francisco).

> **Quick picks**
> - Clean & modern: `AvenirNext-Regular`, `HelveticaNeue`, `Futura-Medium`
> - Classic reading: `Georgia`, `Baskerville`, `Palatino-Roman`, `Cochin`
> - Newspaper feel: `TimesNewRomanPSMT`, `HoeflerText-Regular`
> - Humanist serif: `IowanOldStyle-Roman`, `Optima-Regular`
> - Monospace: `Menlo-Regular`, `CourierNewPSMT`
> - Decorative heading: `Zapfino`, `SnellRoundhand`, `Copperplate`
